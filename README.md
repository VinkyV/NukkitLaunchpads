# NukkitX Launchpads
Simple launchpad plugin

### Config
```yaml
launchpads:
  blocks:   # block ids
    - 152   # redstone_block
    - 57    # diamond_block
  multiple: # launch multiplier
    - 2     # for redstone_block
    - 4     # for diamond_block
```

### Build plugin
```
mvn clean package
```
